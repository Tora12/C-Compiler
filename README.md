## Getting Started

### Dependencies

* g++ version >11.2.0
* bison version >3.8.2
* flex version >2.6.4

### Installing

* Download c-.tar file
* Run tar command
```
$ tar -xvf c-.tar
```

### Executing program

```
$ c- filename.c-
$ cat filename.c- | c-
$ c- < filename.c-
```
